# **App Name**: ServiceFlow

## Core Features:

- Task Creation: Create tasks with priority levels (High, Medium, Low) and assign them to staff members, linking them to existing or new customer profiles. Include task creation and resolution dates.
- Customer Management: Manage customer profiles, including storing relevant information like Aadhaar, PAN, and passport details (encrypted for security).
- Invoice Upload: Upload invoices and extract data such as invoice number, customer name, date, and Aadhaar number. Only name and date are mandatory.
- Reporting & Analytics: Generate reports on task completion rates, staff performance, and invoice summaries.
- Smart Task Assignment Tool: AI-powered tool that analyzes task requirements and automatically suggests the most suitable staff member based on their skills, workload, and availability. The AI will consider the task's priority and due date in its recommendations.
- Employee Creation: Allow admin to create and manage employee accounts.

## Style Guidelines:

- Primary color: Deep sky blue (#3399FF) to inspire trust and productivity.
- Background color: Very light blue (#F0F8FF), for a clean, non-distracting workspace.
- Accent color: Cyan (#00FFFF), used to highlight key actions and calls to action.
- Body and headline font: 'Inter', a sans-serif for modern, readable interface.
- Use a set of consistent, professional icons to represent different task types, priorities, and customer services.
- Prioritize a clean, organized dashboard layout with clear sections for task management, customer data, and invoice uploads.
- Subtle transitions and loading animations to enhance the user experience without being distracting.
